class btree.sorted_btree¶

A n-ary tree container type for sorted data

The constructor takes 1 argument, the tree’s order. This is an integer that indicates the most data items a single node may hold.

static bulkload()¶

create a sorted_btree from a pre-sorted list (classmethod)

Parameters:
  • data (iterable) – the items to load into the sorted_btree
  • order (int) – the branching order for the sorted_btree
Returns:

a new loaded sorted_btree instance
depth¶

The current depth of the tree (just a single leaf is a depth of 0)

insert¶

insert an object into the sorted_btree

Parameters:
  • obj – the object to insert
Returns:

None
order¶

The tree’s order as passed to the constructor.

remove¶

remove an object from the sorted_btree if found by == comparison

Parameters:
  • obj – the object to remove
Raises :

ValueError if the object is not in the sorted_btree
Returns:

None
split¶

divide the sorted_btree into 2 btrees by a separator value

Parameters:
  • separator – the split point – objects greater than separator go into the left sorted_btree, objects less than go into the right sorted_btree
  • eq_goes_left (bool) – if False then objects that are == separator go into the right subtree, otherwise they go to the left. defaults to True
Returns:

a two-tuple of (left_tree, right_tree). the left_tree is actually the original tree, which has now been modified.
